Each participant is assigned a VM where we have two container databases CDB1 and CDB2 running.
You will be doing the labs as the user oracle (password: oracle)

Location of labs: /u01/HOL/
In this folder, you will find 4 folders each containing the scripts that will walk you through a set of features that we are planning to introduce in the upcoming release of the Oracle database.

Provisioning - Hot Clone,Refresh and Relocate
/u01/HOL/provisioning
Lockdown Profiles
/u01/HOL/lockdown_profiles
Application Containers
/u01/HOL/app_containers

Note the OS password for the oracle user is oracle and the VNC password is welcome.

If you are using VNC please open a terminal to execute the scripts.

Note: While doing the labs you are expected to execute the scripts in sqlplus and go through the flow.


Verify your environment

Login to your VM [oracle/oracle] then source your environment.

[oracle@osc-wkshp ~]$ cd /u01/HOL

[oracle@osc-wkshp HOL]$ source cdb1.env

[oracle@cdb1-osc-wkshp:4 HOL]$ cd app_containers/

[oracle@cdb1-osc-wkshp:4 app_containers]$


[oracle@cdb1-osc-wkshp:4 app_containers]$ ps -ef|grep -i pmon|grep -v grep

oracle 20638 1 0 13:54 ? 00:00:02 ora_pmon_cdb1

oracle 21112 1 0 13:59 ? 00:00:02 ora_pmon_cdb2

You should see two pmon processes. If you do not see two databases running you can start them by executing /u01/HOL/startdb.sh
change directory to /u01/HOL and type ./startdb.sh
